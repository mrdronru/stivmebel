## Зачем нужна кромка

Когда плиту разрезают, на срезе обнажается внутренняя структура — прессованная стружка. Она как губка: впитывает влагу, разбухает, рассыпается. Кромка — это полоска материала, которую приклеивают на торец, чтобы защитить плиту.

Без кромки мебель не служит — это не украшение, это необходимость.

## Виды кромки

**Кромка ПВХ** — самая распространённая. Делается из поливинилхлорида, бывает в тысячах цветов и текстур. Гибкая, прочная, влагостойкая.

**Кромка АБС** — из акрилонитрилбутадиенстирола. Более экологична, чем ПВХ (нет хлора). Такая же прочная. Сейчас ABS всё активнее вытесняет ПВХ.

**Кромка меламиновая** — тонкая бумажная плёнка с клеем. Самая дешёвая. Используется для невидимых внутренних срезов — полок, перегородок.

**Кромка деревянная** — массив или шпон. Применяется в мебели из массива для сохранения единого стиля.

## Толщина имеет значение

| Толщина | Где применяется |
|---|---|
| 0.4 мм | Невидимые внутренние торцы |
| 1 мм | Стандарт для большинства мебели |
| 2 мм | Видимые внешние торцы, фасады |
| 3 мм | Столешницы, подоконники |

Чем толще кромка — тем прочнее и незаметнее стык. На дешёвой мебели часто ставят 0.4 мм везде — экономия, которая выходит боком.

## Как проверить качество кромки

Хорошая кромка:
- Плотно прилегает по всей длине без зазоров
- Не отходит на углах
- Аккуратно зашлифована заподлицо с плоскостью плиты
- Цвет совпадает с декором плиты

Плохая кромка:
- Видна полоска клея по периметру
- Угол загнут или отклеивается
- Торчит над поверхностью (не зашлифована)
- Не совпадает по цвету
